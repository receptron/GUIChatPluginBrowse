## Plugin Overview

Web browsing plugin for GUI Chat applications. Fetch and display web content within the chat.

## Common Guidelines

## Plugin-Specific Notes

### Features
- Fetch web pages
- Extract and display content
- Handle various content types

### Dependencies
- `gui-chat-protocol`: Core protocol for GUI Chat plugins
